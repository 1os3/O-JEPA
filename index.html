Abstract 摘要

当前端到端自动驾驶模型深陷于繁重的显式几何转换（如 LSS）、高昂的逐 Token 自回归解码（如 LLM）以及极端的算力依赖中。 为打破这一僵局，本文提出了一种具有极简架构与深层认知能力的顶尖端到端模型—— O-JEPA (Oscillating-JEPA)。本模型抛弃了所有手工几何标定与像素级生成，全程在紧凑的隐空间内运行， 具有极佳的参数效率，可在单张普通消费级显卡上完成全流程训练。

核心创新包括：

  1. 无手工投影的隐式 3D-BEV：摒弃 LSS / IPM 等手工设计的视锥投影管线——相机内外参仅一次性用于计算每个像素的射线方向，由 3D RoPE 注入图像 Token，再经交叉注意力让模型自主学习 2D→3D 的隐式空间对应；训练期通过 LiDAR 密集射线采样与特征一致性蒸馏几何先验。
  2. 振荡分层表征模型 (O-HRM)：采用共享权重的时空双频周期振荡网络，实现高层语义与低层世界状态的跨尺度耦合。
  3. 并行涌现式思维链与末端分化 (WTA-Branching)：引入若干高度概括的语义 Token 并在网络末端分化出多个并行预测头，利用 Winner-Takes-All (WTA) 与最大熵正则化，在仅有确定性监督数据的场景下自发涌现出多模态的未来预测能力——整个思维链在一次前向中并行产出，无需任何逐 Token 文本解码。

1 Introduction 引言

端到端自动驾驶（E2E AD）正从基于模块化的感知-决策管线向统一人工智能演进。然而，现有范式存在三大致命缺陷：

① 几何标定依赖

主流模型高度依赖精准的相机内外参以构建视锥与 BEV，这在跨车型、跨传感器布局部署时带来灾难性的泛化难题。

② 多模态未来坍缩

人类驾驶面临多分支的未来可能（变道、跟车、急刹），但真实数据往往仅含单一确定性操作，导致模型陷入 Mode Collapse。

③ 算力 / 可解释缺失

引入 VLM/LLM 虽带来丰富思维链，但庞大参数量与逐 Token 自回归解码的时延使纯车端部署遥不可及。

为此，我们提出了统一且干净的 O-JEPA 架构。受大脑层级预测编码机制 (Hierarchical Predictive Coding) 与隐式神经场 (NeRF) 思想启发， O-JEPA 实现了一种低成本、强解释、高动态容量的全新基座。

2 Methodology 核心架构方法

O-JEPA 摒弃了传统的解码器重建管线，整个模型被定义为一个序列到序列（Seq2Seq）的隐空间映射过程： O-HRM 引擎在固定步数内对隐状态做共享权重的自我迭代精化（fixed-step latent self-refinement， 可视为隐空间内的定步数自回归），而输出层不进行任何逐 Token 解码—— 所有未来预测、轨迹与控制指令均由并行分支头在单次前向中一次性产出。 全局特征被投影至轻量化友好的统一维度中。

2.1 视觉编码与免标定隐式 BEV 场 Calibration-Free Implicit BEV

传统的基于 LSS 等方法的 BEV 构建需要显式的视锥投影和几何运算。我们提出一种纯注意力机制的连续三维表征空间构建方法：

  • 深层视觉先验：以 DINOv3 强自监督表征模型作为视觉骨干，提取高维语义特征 $\mathcal{F}_{img}$。骨干随主干端到端参与训练以适配驾驶域分布，无需借助任何手工的 2D 检测/分割监督。
  • 3D RoPE 射线位置编码：为图像平面的每个像素定义基于自车坐标系的射线方向向量，并通过 3D 旋转位置编码（3D RoPE）将射线几何信息注入图像 Token。
  • 全注意力投影：初始化一定规模的可学习 BEV Tokens $\mathcal{T}_{bev}$（位置编码定义为可学习的隐式向量）。以 $\mathcal{T}_{bev}$ 为 Query，带射线方向信息的 $\mathcal{F}_{img}$ 为 Key/Value，模型通过交叉注意力自主学习 2D→3D 的空间对应关系——全过程不引入任何手工设计的视锥/几何投影（如 LSS、IPM、depth-lift），相机内外参仅在生成 3D RoPE 射线方向时一次性使用。
  • LiDAR 密集射线采样蒸馏（仅训练期 Teacher）：在训练阶段沿 LiDAR 射线密集采样，利用网络预测三维空间特征 $\mathcal{F}_{3D}$，引入特征一致性约束，使 $\mathcal{F}_{3D}$ 逆投影后必须与图像特征强对齐。这迫使隐空间在无任何手工投影管线的前提下，自发构建出具几何与语义一致性的连续三维表征。

关于"免标定 / Calibration-Free"的澄清：本文使用该术语 特指摒弃 LSS / IPM 等手工设计的视锥投影几何，而非"完全不需要相机内外参"。 内外参依然用于一次性计算每个像素的 3D 射线方向（生成 3D RoPE 编码），但不再参与 任何手工几何变换——所有 2D→3D 的空间对应关系完全交由交叉注意力自学。这也正是 O-JEPA 跨车型 / 跨传感器布局可大幅降低适配成本的根本原因。

2.2 序列组装与状态三重注入 Sequence Assembly & Triple-Injection

进入认知引擎的输入序列经过精心设计，由三大类表征构成：

  1. 世界环境 Token：上述的 $\mathcal{T}_{bev}$，负责压缩并承载多帧时空的历史隐式特征。
  2. Visual CoT 语义槽：引入多个可学习 Prompt Tokens，细粒度地锚定不同的思维维度（例如：[场景]、[交规]、[潜在危险]、[下一步推演]、[本车决策]、[决策后果]）。
  3. 状态三重注入：对于导航指令、自车运动学状态和目标点信息 $S_{state}$，为防止高维视觉 Token 将其淹没，设计了强化的 Triple-Injection：
    • 独立 Token 注入：作为显式上下文 Token 拼接于序列。
    • 特征强行拼接：在通道维度将状态表征与 BEV Token Concatenate（结合降维以维持主干维度不变）。
    • AdaLN 调制：由状态特征生成全局仿射 $\gamma, \beta$，在 LayerNorm 阶段执行全局调制。

2.3 振荡分层表征网络 Oscillating HRM Engine

传统时空网络通过不断堆叠串行的 Transformer 层来获取深层感受野，导致参数量线性膨胀。O-JEPA 提出采用极其精简的核心网络构建高频与低频耦合的双周期振荡网络（O-HRM）。

  • 双重周期机制：模型在时间与逻辑轴上被划分为多个高层周期（宏观语义/低频），每个高层周期内部嵌套多个低层周期（微观状态推演/高频）。
  • 权重共享 + AdaLN 身份识别：所有周期的前向计算均由同一个共享权重的网络模块执行；由“高层步数 / 低层步数 / 层级类型”生成嵌入向量，通过 AdaLN 注入网络，使模型在“宏观战略深思”与“微观战术推演”间振荡切换。
  • 双向交互与全序列感知：高层（抽象策略 / 宏观语义）与低层（时空流转 / 未来环境演变）均直接处理全局序列；高低层交替之际通过双向交叉注意力（Bi-CA）同步特征，实现宏观认知与微观感知的完美闭环。

关于"自回归"的澄清：O-HRM 的振荡迭代会反复读入自身上一周期的隐状态， 在严格定义上属于隐空间内的定步数自回归（latent fixed-step autoregression）——但其步数预先确定、所有迭代共享同一组参数、且不向外输出任何 Token， 因此与 GPT 类逐 Token、变长度、串行解码的语言自回归在复杂度、可并行性与端侧延迟上有本质差异。 全文中的"非自回归 / Token-AR Free"措辞均指输出层不进行逐 Token 解码。

2.4 末端分化与 WTA 多概率预测 Terminal Branching & Winner-Takes-All

在 O-HRM 完成主干特征提取后，为了在人类给定的单一直觉驾驶数据（确定性真值）中逼迫模型涌现“多模态概率思考”，引入末端分化策略。

  • 并行分支头：主干网络末端分裂出 $K$ 个独立轻量级 Head；每个 Head 并行输出未来多帧的隐状态预测序列 $\hat{\mathcal{S}}_k$ 与对应语义预测 Token $\hat{\mathcal{C}}_k$，并各自附带一个轻量回归子模块输出该 Head 自身的置信度 $p_k$，组合成先验分布 $P=[p_1,\dots,p_K]$。$P$ 由 NLL 损失独立监督学习（见 §3），代表"该 Head 在当前场景下应被信任的程度"。
  • WTA 梯度截断：动态寻找距离真值隐状态最近的“胜出”分支 $k^{*}$（Winner）。
    • 反向传播时，仅允许 $k^*$ 分支回传环境与语义预测梯度。
    • 对其余未胜出分支，截断其预测特征方向的梯度传播。
  • NLL 与不确定性建模：使用负对数似然损失惩罚未胜出分支的先验置信度分布。
  • 防止模式坍缩：引入最大熵正则化 $\mathcal{L}_{Ent}=\sum p_k\log p_k$，强制各 Head 维持均等探索概率，促使其自发涌现出截然不同的未来世界线（专职变道、跟车或避让等 Head）。

2.5 自适应动作解码器 Adaptive Action Decoder

  • 置信度调制的交叉注意力：Decoder 以可学习查询向量为 Query，将 $K$ 个 Head 的预测特征 $\{\hat{\mathcal{S}}_k, \hat{\mathcal{C}}_k\}$ 拼接为 Key / Value。先验置信度 $P$ 不直接对预测特征做加权求和，而是作为偏置项调制注意力分数——形式上即 $$ \mathrm{Attn}(Q,K,V) = \mathrm{softmax}\!\left(\tfrac{QK^{\top}}{\sqrt{d}} + \log P\right)V $$ （或等价的乘性调制）。这样可信的 Head 自然主导特征抓取，可疑的 Head 被注意力机制天然抑制，而 Q 仍保留在不同 Head 之间做内容相关取舍的能力。
  • 直接回归：基于聚合后的统一特征，直接确定性输出短期轨迹规划点与底盘级控制指令（转向、油门、刹车）。

3 Training Objective 训练策略与损失函数

模型支持大规模的自监督学习（SSL）为主，并辅以少量强监督的微调对齐（SFT）。

Phase 1: 隐空间联合嵌入预训练 SSL

通过屏蔽部分未来的感知输入，利用 WTA 机制让模型在纯隐空间预测未来：

隐世界预测损失 · WTA-MSE
$$ \mathcal{L}_{world} = \lVert \hat{\mathcal{S}}_{k^{*}} - \mathcal{S}_{gt} \rVert_2^2 $$
多模态探索损失 · NLL + Entropy
$$ \mathcal{L}_{branch} = -\log(p_{k^{*}}) + \lambda_{ent}\sum_{k} p_k \log p_k $$
隐式三维特征一致性损失 · LiDAR Teacher
$$ \mathcal{L}_{bev} = 1 - \mathrm{CosineSimilarity}\!\left(\mathrm{Proj}(\mathcal{F}_{3D}),\, \mathcal{F}_{img}\right) $$

Phase 2: 语义蒸馏与控制对齐 SFT

无需在车端实时运行高耗能的语言模型。利用云端大语言/多模态模型离线生成驾驶思维链，并通过文本嵌入模型提取为固定维度的真实语义向量 $E_{cot}$。

语义思维链蒸馏 · Visual CoT Alignment
$$ \mathcal{L}_{sem} = \sum_{i}\bigl(1 - \mathrm{CosineSimilarity}(\hat{\mathcal{C}}_{k^{*}}[i],\, E_{cot}[i])\bigr) $$

控制与轨迹模仿采用标准回归损失（如 L1 Loss）与监督数据对齐：$\mathcal{L}_{act}$。

总优化目标
$$ \mathcal{L}_{Total} = \mathcal{L}_{world} + \mathcal{L}_{branch} + \lambda_1 \mathcal{L}_{bev} + \lambda_2 \mathcal{L}_{sem} + \lambda_3 \mathcal{L}_{act} $$

4 Efficiency Analysis 架构极度轻量化分析

O-JEPA 严格遵循极致工程轻量化原则，其单卡可训的底层逻辑在于巧妙的架构设计而非暴力算力堆叠：

① 强视觉先验 + 端到端可训

以 DINOv3 强自监督表征为骨干，提供丰富的语义先验；骨干随主干一起端到端训练以适配驾驶域分布，无需任何手工 2D 监督。

② 空间转换的低代价

无手工投影的交叉注意力 BEV 编码器仅依赖单层注意力即可完成 2D→隐式 3D 聚合，避免高耗能 3D 卷积，亦省去 LSS 类管线的工程负担。

③ 认知引擎的极致复用

O-HRM 振荡网络完全共享权重，时间步上展开多次循环达到极深推理深度，物理参数量仅相当于极少数层的标准 Transformer——以时间复杂度换取空间复杂度。

④ 无重型解码器

全程不进行图像级高分辨率像素重建（摒弃 Diffusion / GAN），亦不进行逐词自回归生成，激活内存与反向计算图被压缩到极致。

这些极简且纯粹的设计，使得整个模型的主体可训练参数被控制在极为轻量级的范畴。研究者能够轻易地在普通单张消费级显卡上，采用可观的 Batch-Size 运行并行训练；而在车端推理时，纯隐空间的矩阵乘法能够将系统延迟（Latency）压缩至极低水位。

5 Conclusion 结论

O-JEPA 重新定义了端到端自动驾驶的基础范式。通过无手工投影的隐空间神经场（仅以内外参生成 3D RoPE 射线方向，由交叉注意力自学几何对应）大幅降低了跨车型 / 跨传感器布局的适配成本； 通过共享振荡网络（O-HRM）以极低的参数代价实现了深不可测的时空表征能力； 通过WTA 多概率分支以单次前向、无逐 Token 解码的极低成本涌现出大模型才具备的“思维链与推演”能力。 O-JEPA 严格遵守了严苛的算力与显存约束，展现了作为下一代兼具高智能与高解释性的量产级自动驾驶基座的巨大潜力。
